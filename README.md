# Finanças da Família

Aplicativo familiar para registrar movimentações financeiras em poucos segundos.

## Beta Familiar 0.5

Objetivo: permitir que Augusto e Izabela registrem despesas, receitas, parcelas e contas fixas no dia a dia, com dashboard simples do mês.

## Publicação

Este projeto está preparado para GitHub Pages.

## Backend

O backend fica no Google Apps Script. Os arquivos estão na pasta `apps-script/` e devem ser copiados para o projeto Apps Script vinculado à planilha.
