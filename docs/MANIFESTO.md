# Manifesto do Projeto

Este aplicativo nasceu para resolver um problema da nossa própria família.

Depois de testar diversos aplicativos financeiros, percebemos que todos exigiam muitos passos para registrar uma simples despesa. Como consequência, abandonávamos o controle financeiro depois de alguns dias.

A primeira ideia foi criar um grupo no WhatsApp onde registraríamos todos os gastos para depois passá-los manualmente para uma planilha.

Então surgiu uma pergunta:

> Por que não criar um aplicativo que faça isso automaticamente?

Foi assim que nasceu este projeto.

Nosso objetivo nunca foi criar o aplicativo financeiro mais completo.

Nosso objetivo é criar o aplicativo financeiro que as pessoas realmente continuem usando.

Se registrar uma movimentação levar mais de alguns segundos, estamos fazendo algo errado.

Se o usuário precisar aprender como usar o aplicativo, estamos fazendo algo errado.

O usuário escreve.  
O aplicativo interpreta.

Sempre construiremos este projeto pensando primeiro na nossa família.

Se um dia ele ajudar outras famílias, isso será consequência, nunca o objetivo inicial.

## Princípios

1. Registrar uma movimentação deve levar menos de 5 segundos.
2. O usuário escreve. O aplicativo interpreta.
3. O aplicativo pergunta apenas quando realmente precisa.
4. Uma compra parcelada é registrada apenas uma vez.
5. Bons hábitos são mais importantes do que dezenas de funcionalidades.
6. Cada sprint deve resolver um único problema.
7. O projeto cresce junto com o Augusto e a Izabela.
8. Terminamos antes de expandir.
9. Boas ideias não precisam ser implementadas imediatamente.
10. O usuário sempre vence o planejamento.

## Frase-guia

> Este aplicativo nasceu para resolver o problema de uma família. Se ele resolver o problema de outras também, será consequência do cuidado com que foi construído.
